# 智慧運勢系統 - 三元合一命理平台

一個結合**易經占卜**、**紫微斗數**和**西洋星座**的現代化智慧運勢分析系統，為您提供深度個人化的每日指導。

![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![Language](https://img.shields.io/badge/language-JavaScript-yellow.svg)

## 主要特色

### 三元合一分析系統

- **易經智慧** - 完整的 64 卦 384 爻辭解釋系統
- **紫微斗數** - 基於天機星、巨門星的命盤分析
- **西洋星座** - 即時星象能量整合
- **AI 整合** - 智能化的個人化建議生成

### 五層級智慧解析

1. **大吉大利** (元吉、大吉、亨) - 黃金機會指導
2. **順遂吉利** (吉、利) - 有利行動建議
3. **謹慎考驗** (凶、厲、吝) - 謹慎策略建議
4. **平和安全** (無咎、無悔) - 穩定維護方法
5. **觀察洞察** - 深度洞察和耐心指導

### 用戶體驗

- **每日幸運色** - 基於卦象的色彩能量
- **動態主題** - 隨運勢變化的視覺體驗
- **格式化輸出** - 結構化的建議呈現
- **多功能導航** - 運勢、美食推薦等模組

## 快速開始

### 線上體驗

直接訪問：[智慧運勢系統](https://ymcglu.github.io/web/)

### 本地運行

```bash
# 克隆項目
git clone https://github.com/ymcglu/web.git

# 進入項目目錄
cd web

# 使用本地服務器運行
python -m http.server 8000
# 或使用 Node.js
npx serve .

# 在瀏覽器中打開
open http://localhost:8000
```

## 技術架構

### 核心系統

- **前端**: 純 JavaScript (ES6+)、HTML5、CSS3
- **API 整合**: Aztro 星座 API
- **數據結構**: 完整的易經卦象與爻辭資料庫
- **響應式設計**: 支援桌面和移動設備

### 關鍵功能模組

#### 1. 易經占卜系統 (`iChingData`)

```javascript
// 64卦完整資料庫
const iChingData = {
    1: { name: "乾", lines: ["潛龍勿用。", ...] },
    // ... 64卦詳細資料
};
```

#### 2. 爻辭解釋系統 (`yaoCiExplanations`)

```javascript
// 384條爻辭白話解釋
const yaoCiExplanations = {
  "潛龍勿用。": "時機尚未成熟，就像潛伏的龍...",
  // ... 完整解釋資料庫
};
```

#### 3. 五行色彩系統 (`guaColors`)

```javascript
// 八卦對應色彩能量
const guaColors = {
  乾: { name: "天界金", hex: "#E5E4E2" },
  // ... 完整色彩對應
};
```

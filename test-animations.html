4. Prefers-Reduced-Motion Test

To test reduced motion:

  1. Open browser developer tools
  2. Go to Console
  3. Run: document.body.style.setProperty('--animation-duration', '0.01ms')
  4. Or enable "Reduce motion" in your OS accessibility settings
